# 🎓 Student Management System

A modern multi-organization Student Management System built with Python, Flask, PostgreSQL, Bootstrap, Cloudinary, and Render.

## 🌐 Live Demo

https://studentmanagementsystem-wcec.onrender.com

---

## 🚀 Features

### Authentication & Security

* Admin Login
* Staff Registration
* Teacher Registration
* User Approval System
* Role-Based Access Control
* Password Management

### Student Management

* Add Students
* Edit Students
* Delete Students
* Search Students
* Student Profile View
* Student Photo Upload

### Organization Management

* Multi-Organization Architecture
* Organization Isolation
* Organization Settings
* Organization Deletion

### User Management

* User Approval Workflow
* Staff Management
* Teacher Management
* User Status Tracking

### Cloud & Deployment

* Cloudinary Image Storage
* PostgreSQL Database
* Render Deployment
